Stock Price Prediction: NETFLIX (Python)

Used pandas datareader option to import Netflix stock data over the last ten years. Yahoo Api was used to obtain the dataset. The objective of the project was to understand how models such as LSTM/ ARIMAX/ Auto ARIMA works and thus to predict the stock prices for next 2 months and then compare with the actual prices in future. Libraries used Scikit Learn, Matplotlib, Tensorflow, ARIMAX. Keras, Dense, Sequential models were used to predict the future prices Key metrics such as ADF, P-value was implemeted to test normality of data Data preprocessing was done using MinMaxScaler and then the data set was split into training and testing datasets LSTM was implemented and a comparison between actual prices and predicted prices was done.
